There is the idea that everything is one, a holistic whole, but that is the reaction to a failing exploration quest to the indivisable element of our existence. We want to break everything down into more and more complex models of unique but ever more generic building blocks and relations between them. But these building blocks did not exist until we defined them and isolated them. They are destined to be abstract and intangible in the most fundamental sense. It is therefore that the relations which are already intangible that are more existing. There is no beginning or end but there is a way. There is no truth goal, but there is a striving. There are no armies but there is a battle, no lovers but love, no hateful objects but hate, no presses but a force.

Only of these things can we speak of any certainty and only these things we can truly experience.
